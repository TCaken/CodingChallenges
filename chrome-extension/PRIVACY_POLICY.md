# Privacy Policy — Coding Challenges New Tab

**Last updated:** June 23, 2026

## Overview

"Coding Challenges New Tab" is a Chrome extension that replaces your new tab page with a custom dashboard showing the current time and open pull requests from the CodingChallengesFYI GitHub repository.

## Data Collection

This extension does **not** collect, store, or transmit any personal data.

## Network Requests

This extension makes requests to:
- **GitHub API** (`https://api.github.com`) — to fetch open pull requests from the public CodingChallengesFYI/SharedSolutions repository.

No authentication is required for this request. No user data is sent to GitHub or any other third party.

## Storage

This extension does not use local storage, cookies, or any other form of data persistence.

## Permissions

- `host_permissions` for `https://api.github.com/*` — required to fetch public pull request data from GitHub's API.

## Changes

If this privacy policy changes, it will be updated in this document and the extension listing.
